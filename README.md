# 부산경상대학교 소개 웹사이트

HTML, CSS, Bootstrap을 이용하여 제작한 부산경상대학교 소개 웹사이트입니다.

## 프로젝트 소개

이 프로젝트는 부산경상대학교의 학과, 대학 건물, 역사, 교수 및 학생 정보를 소개하기 위해 제작한 웹사이트입니다.

## 주요 기능

- 학과 안내
- 대학 건물 및 학과 소개
- 대학 역사 소개
- 교수 및 학생 소개
- Contact(문의) 페이지
- 반응형 Bootstrap UI

## 개발 환경

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Font Awesome

## 실행 방법

1. 저장소를 다운로드합니다.
2. `index.html`을 실행합니다.
3. 또는 GitHub Pages를 이용하여 접속합니다.
